## 归档原因

这些文档被归档的主要原因：

1. **临时性文档**: 用于记录特定功能的实现过程，功能完成后已无参考价值
2. **已合并内容**: 内容已整合到其他主要文档中
3. **过时信息**: 记录的是旧版本的实现方式，已被新方案替代
4. **历史记录**: 作为项目发展历程的记录，但不再作为开发参考

## 使用建议

- 如需了解某个功能的实现历程，可参考对应的归档文档
- 日常开发请参考 docs 目录下的主要文档
- 归档文档不保证与当前代码库同步
